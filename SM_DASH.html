🤖 EXECUTAR AI INSIGHTS

PASSO 1 — CONFIGURAR PROVEDOR DE IA
Opção A — Ollama (Local, gratuito):
Instale o Ollama em ollama.com e execute:
ollama pull llama3.2:1b Opção B — Gemini (Nuvem):
Crie o arquivo C:\Dashboard\.env com:
DEFAULT_AI_PROVIDER=gemini
GEMINI_API_KEY=SUA-CHAVE-AQUI
Chave gratuita em: aistudio.google.com/app/apikey
PASSO 2 — INSTALAR DEPENDENCIAS (1 vez)
pip install pandas openpyxl lxml
PASSO 3 — EXECUTAR
Todos os 6 agentes agora:
python C:\Dashboard\smd_agent.py
Apenas 1 agente:
python C:\Dashboard\smd_agent.py --agent ops
python C:\Dashboard\smd_agent.py --agent predictive
python C:\Dashboard\smd_agent.py --agent improvement
python C:\Dashboard\smd_agent.py --agent market
python C:\Dashboard\smd_agent.py --agent qa
python C:\Dashboard\smd_agent.py --agent triage
Automatico (07h e 15h todo dia):
python C:\Dashboard\smd_agent.py --schedule
PASSO 4 — VER RESULTADOS
Apos executar, o script gera os JSONs em C:\Dashboard\Resultados\
Recarregue o dashboard para ver os insights atualizados.
O smd_agent.py tambem recria o dashboard com os dados do agente integrados.
DICA — FORÇAR PROVEDOR NA LINHA DE COMANDO
Use --provider para sobrescrever o smd_config.py:
python C:\Dashboard\smd_agent.py --provider ollama
python C:\Dashboard\smd_agent.py --provider gemini
